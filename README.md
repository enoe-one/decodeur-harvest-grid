# 🇬🇧 English Version

## 📖 About the Project

**decodeur-h.g-mini-game** is a personal project built on top of another project that I respect and admire.

The goal of this project is **not to damage or exploit the original game**, but rather to:

- understand how it works internally
- experiment with its mechanics
- explore possibilities such as creating an **aesthetic garden**

This repository mainly exists for **educational and personal exploration purposes**.

## ⚠️ Disclaimer

This project is based on the original work created by **Paul Maxime**.

If the creator of the original project prefers that this repository not exist, **I will remove it immediately upon request**.

- Website: https://paulmaxime.fr/harvest/
